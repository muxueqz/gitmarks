gitmarks
Bookmarklets add one-click functions to a browser or web page.

What

  1. Browse and select text
  2. Click bookmark
  3. Add labels
  4. Manage and Search bookmarks

How

  1. Fork https://github.com/petrosh/gitmarks
  2. Make a commit to build gh-pages: edit `README.md`
  3. Browse to `https://.github.io/gitmarks`
  4. Drag the bookmarklet below on your toolbar

Manage bookmarks in the repository issues page.
